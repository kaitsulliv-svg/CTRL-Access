# CTRL+Access @ DIYode

> **Reducing the Disability Tax Through Community Manufacturing**

CTRL+Access is a community assistive technology program based at **DIYode Community Workshop** in Guelph, Ontario. We design, build, and distribute low-cost adaptive devices for people with disabilities — openly, collaboratively, and with care.

Funded by a **$3,000 micro-grant from Makers Making Change (MMC)**.

## Core Values

- **Accessibility First** — Every decision starts with: is this accessible?
- **Community Before Competition** — We share everything: designs, documentation, failures.
- **Open Source Always** — Every device is documented and released under an open-source license.
- **Documentation Matters** — A device that isn't documented can't be replicated, improved, or trusted.
- **Finished is Better than Perfect** — Ship it, document it, improve it.
